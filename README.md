# Portafolio Profesional - Nassir Santiago Cárdenas Caldas

Este repositorio contiene el código fuente de mi portafolio web personal y hoja de vida interactiva. Diseñado para mostrar de manera clara mis habilidades, formación y proyectos destacados en el área de **Ciencia de Datos** y **Desarrollo Web**.

Puedes ver el portafolio en vivo en:  
👉 **[https://killerdoller.github.io/cv/](https://killerdoller.github.io/cv/)**

---

## 🛠️ Tecnologías Utilizadas

* **HTML5**: Estructura semántica completa para mejorar el SEO y la accesibilidad.
* **Vanilla CSS3**: Diseño personalizado responsivo, variables CSS y transiciones fluidas.
* **Vanilla JavaScript**: Control interactivo de navegación y rastreo de secciones dinámico.

---

## 📂 Contenido del Portafolio

* **Sobre Mí**: Información sobre mi formación actual en Ciencia de Datos en la Universidad Externado de Colombia y mi interés por el modelado estadístico.
* **Proyectos Destacados**:
  * **Uso Problemático de Redes Sociales (PSMU)**: Investigación cuantitativa sobre la influencia del tiempo de uso, la frecuencia de revisión, la edad y el sexo en el uso problemático de redes sociales (incluye archivo PDF adjunto en el repositorio).
  * **RentAI**: Proyecto colaborativo para la búsqueda inteligente de roommates e inmuebles en alquiler.
  * **pokemonML**: Modelo supervisado de Machine Learning para predecir si un Pokémon es legendario a partir de sus estadísticas de combate.
* **Habilidades**: Organizadas por categorías técnicas (Lenguajes, Ciencia de Datos y Tecnologías).
* **Contacto**: Enlaces directos a mi correo electrónico y perfil de GitHub.

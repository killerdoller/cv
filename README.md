# Portafolio Profesional - Santiago Nassir

Este repositorio contiene el código fuente de mi portafolio web personal y hoja de vida interactiva. Diseñado para mostrar de manera clara mis habilidades, experiencia y proyectos destacados como **Desarrollador Full-Stack** y **Tecnólogo Creativo**.

Puedes ver el portafolio en vivo en:  
👉 **[https://killerdoller.github.io/cv/](https://killerdoller.github.io/cv/)**

---

## 🛠️ Tecnologías Utilizadas

* **HTML5**: Estructura semántica completa para mejorar el SEO y la accesibilidad.
* **Vanilla CSS3**: Diseño personalizado responsivo, variables CSS y animaciones fluidas.
* **Vanilla JavaScript**: Control interactivo de navegación y rastreo de secciones dinámico.

---

## 📂 Contenido del Portafolio

* **Sobre Mí**: Una descripción de mi perfil profesional y mi enfoque en sistemas interactivos y de desarrollo web.
* **Proyectos Destacados**:
  * **Gabbanimaniac**: Portal interactivo y comunidad de live-coding con procesamiento de audio en tiempo real.
  * **RentAI**: Aplicación de emparejamiento inmobiliario e inquilinos inteligentes utilizando Next.js y Supabase.
  * **Live-Coding Sound Engine**: Configuración avanzada en SuperCollider y TidalCycles.
* **Habilidades**: Organizadas por categorías técnicas (Frontend, Backend/Datos, Audio/Creative Coding y Herramientas).
* **Experiencia y Educación**: Una línea de tiempo clara con mi historial profesional y formación reciente.
* **Contacto**: Acceso directo para comunicaciones profesionales.

## 💻 Ejecución Local

```bash
# Abrir index.html en tu navegador preferido
```
